## A quick start

1. Create a free account at **chatgpt.com** or **claude.ai**.
2. Pick the one task that eats your week.
3. Brief the AI with the **R·C·A·T·C** recipe (Role · Context · Audience · Tone · Constraints).
4. Use AI to **draft — never to decide**, and review everything before you send.

> Treat AI like a brilliant, eager intern — the better you brief it, the better it works.
